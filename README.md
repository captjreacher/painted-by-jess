# Painted By Jess — website

Static one-page marketing site for **Painted By Jess** — a female-owned & operated
house painting crew serving West & North-West Auckland (based in Kumeū).

## Hosting
- **GitHub Pages** — source: `main` branch, root (`/`)
- **Custom domain:** `painted-by-jess.staging.maximisedai.com` (see `CNAME`)
- `.nojekyll` tells Pages to serve files as-is (no Jekyll build)

## Files
| File | Purpose |
|------|---------|
| `index.html` | The complete site — self-contained (inline CSS/JS, Google Fonts + images over HTTPS) |
| `CNAME` | Custom domain for GitHub Pages |
| `.nojekyll` | Serve static files without Jekyll processing |
